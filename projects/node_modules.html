Node Modules

Canvas Watchfaces

I created this node module to be able to show off some of my watch face designs on my personal website. The problem: all my Android Watch watch faces were created via Watchmaker Pro and are saved in a format unreadable by browsers. The solution: I created a node module that takes .watch files, parses them, and outputs a javascript file that draws a nearly perfect facsimile as an HTML Canvas element.

LIRI Bot

This node module was a homework assignment for the Columbia Full-Stack Web Development Bootcamp as an entry into API calls using node. It interfaces with Reddit, Spotify and OMDB APIs.
